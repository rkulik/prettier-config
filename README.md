# @rkulik/prettier-config

René's Prettier configuration.

## Usage

### Install

```bash
pnpm add -D prettier @rkulik/prettier-config
```

### Configure

Once installed, reference it in your `package.json`:

```json
{
  "prettier": "@rkulik/prettier-config"
}
```
